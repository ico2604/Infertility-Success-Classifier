# Infertility Success Classifier

불임/난임 시술 데이터를 기반으로 **임신 성공 여부**를 예측하는 프로젝트입니다.  
본 프로젝트는 단일 모델 성능 개선뿐 아니라, **도메인 피처 엔지니어링**, **subgroup expert 모델링**, **팀원 간 앙상블**을 통해 최종 성능을 높이는 것을 목표로 진행되었습니다.

---

## 1. Project Overview

### 문제 정의
불임/난임 시술 환자의 임상 정보, 시술 방식, 난자/정자 출처, 배아 생성/이식 관련 정보를 활용하여  
**해당 시술이 임신 성공으로 이어질 확률**을 예측하는 이진 분류 문제입니다.

### 최종 목표
- **임신 성공 여부를 최대한 정확하게 예측**
- 전체 데이터뿐 아니라, 특히 예측이 어려운 **이식군(transfer)** / **동결 배아이식군(frozen transfer)** 에서의 성능 개선
- 개별 모델 최적화뿐 아니라 **팀원 모델 간 앙상블을 통해 최종 Public AUC 최대화**

### 평가 지표
- **AUC (Area Under the ROC Curve)**

---

## 2. Dataset

- `train.csv`: `(256,351, 69)`
- `test.csv`: `(90,067, 68)`
- Target:
  - `임신 성공 여부`
  - 0 = 실패, 1 = 성공
- 양성 비율: **25.8%**

### 주요 변수 예시
- 시술 당시 나이
- 시술 유형 / 특정 시술 유형
- 배란 자극 여부 / 배란 유도 유형
- 불임 원인 관련 변수
- 총 시술 횟수 / 임신 횟수 / 출산 횟수
- 총 생성 배아 수 / 이식된 배아 수 / 저장된 배아 수
- 난자 출처 / 정자 출처
- 동결 배아 사용 여부 / 신선 배아 사용 여부
- 배아 이식 경과일 / 배아 해동 경과일 등

## 3. Approach

프로젝트는 크게 다음의 흐름으로 진행되었습니다.

### 3-1. 전처리
- 문자열 결측값 → `Unknown`
- 수치형 결측값 → `0`
- 시술 횟수, 임신/출산 횟수 → 숫자형 파생 컬럼 생성 (`_num`)
- 연령 구간 → 숫자형 파생 컬럼 생성 (`_num`)
- `Y/N`, `예/아니오`, `True/False` 등 boolean 컬럼 → `0/1` 정규화
- CatBoost에 범주형 컬럼 직접 입력

### 3-2. 피처 엔지니어링
- 시술 이력 기반 비율 피처
  - 전체임신률, IVF임신률, 임신유지율 등
- 배아/난자 효율 피처
  - 배아생성효율, 배아이식비율, 배아저장비율, 수정 성공률 등
- 연령 기반 피처
  - 나이 구간, 고령/초고령 플래그, 나이×실패횟수 상호작용
- frozen transfer 관련 피처
  - thaw_to_transfer_ratio, frozen_day_interaction 등
- 도메인 피처
  - effective_oocyte_age, transferable_embryos, thaw_survival_proxy 등

### 3-3. 모델링
- CatBoost
- XGBoost
- LightGBM
- OOF 기반 앙상블

### 3-4. Subgroup Expert 전략
전체 데이터를 하나의 모델로만 학습하는 대신,
- **Global 모델**
- **Transfer expert**
- **Frozen expert**

를 분리해 학습하고, subgroup별로 다른 가중치로 최종 예측값을 생성했습니다.
