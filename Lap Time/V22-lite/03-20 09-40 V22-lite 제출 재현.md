# V22-lite 제출 재현

이 폴더는 `v22-lite 2-seed` 최고 점수 버전을 독립 실행 가능하게 묶은 아카이브입니다.

## 포함 내용
- `v22_lite_notebook.py`
  - v22-lite 2-seed expert 모델 학습 코드
- `run.py`, `run.bat`
  - 패키지 내부 파일만으로 학습 실행
- `data/`
  - `train.csv`, `test.csv`, `sample_submission.csv`
- `npy/`
  - 외부 global blend용 `v17/v19` OOF/test npy
- `result_v22_lite/`
  - 실제 학습 산출물 보관본
- `ensemble_inputs/`
  - 최종 rank ensemble 재현에 필요한 XGB/LGB 입력 CSV
- `make_rank_ensemble.py`
  - 최종 제출 CSV 재생성 스크립트
- `rank_ensemble_v22_lite_2seed_champ_weight.csv`
  - 실제 리더보드 제출본

## 바로 실행하는 방법

### 1. v22-lite 학습 실행
```powershell
python run.py
```

또는

```powershell
run.bat
```

학습이 끝나면 `result_v22_lite/` 아래에 `oof_v22_lite_final.npy`, `test_v22_lite_final.npy`가 생성됩니다.

### 2. 최종 rank ensemble 제출본 재현
```powershell
python make_rank_ensemble.py
```

기본 실행 시 아래 파일을 자동으로 사용합니다.
- `ensemble_inputs/xgb_v2_reg_relax_oof_predictions.csv`
- `ensemble_inputs/xgb_v2_reg_relax_submission.csv`
- `ensemble_inputs/lightgbm_v1_lightgbm_baseline_v1_oof_predictions.csv`
- `ensemble_inputs/lightgbm_v1_lightgbm_baseline_v1_submission.csv`
- `result_v22_lite/oof_v22_lite_final.npy`
- `result_v22_lite/test_v22_lite_final.npy`
- `data/sample_submission.csv`

기본 출력 파일:
- `rank_ensemble_v22_lite_2seed_champ_weight_rebuilt.csv`

## 검증 기준
- `python make_rank_ensemble.py` 실행 후 생성된 rebuilt CSV가
  `rank_ensemble_v22_lite_2seed_champ_weight.csv`와 동일해야 합니다.
- 확인 항목:
  - 행 수 동일
  - `ID` 동일
  - `probability` 값 동일

## 참고
- 최종 rank ensemble 가중치:
  - XGB `0.16`
  - LGB `0.04`
  - CAT `0.80`
- `make_rank_ensemble.py`는 기본적으로 위 제출본과 자동 비교까지 수행합니다.
