# V22-lite 제출 재현

이 폴더는 실제 리더보드 제출본 `v22-lite 2-seed 3모델 rank ensemble` 보관본입니다.

## 포함 파일
- `rank_ensemble_v22_lite_2seed_champ_weight.csv`
  - 실제 제출한 CSV 보관본
- `make_rank_ensemble.py`
  - 제출 CSV를 다시 생성하는 재현 스크립트

## 필요한 입력 파일
- XGB OOF: `xgb_v2_reg_relax_oof_predictions.csv`
- XGB submission: `xgb_v2_reg_relax_submission.csv`
- LGB OOF: `lightgbm_v1_lightgbm_baseline_v1_oof_predictions.csv`
- LGB submission: `lightgbm_v1_lightgbm_baseline_v1_submission.csv`
- CAT OOF: `oof_v22_lite_final.npy`
- CAT test: `test_v22_lite_final.npy`
- sample submission: `sample_submission.csv`

## 가중치
- XGB: `0.16`
- LGB: `0.04`
- CAT: `0.80`

## 실행 예시
```powershell
python make_rank_ensemble.py `
  --xgb-oof "C:\Users\mal03\Desktop\레퍼런스\xgb_v2_reg_relax_oof_predictions.csv" `
  --xgb-sub "C:\Users\mal03\Desktop\레퍼런스\xgb_v2_reg_relax_submission.csv" `
  --lgb-oof "C:\Users\mal03\Desktop\레퍼런스\lightgbm_v1_lightgbm_baseline_v1_oof_predictions.csv" `
  --lgb-sub "C:\Users\mal03\Desktop\레퍼런스\lightgbm_v1_lightgbm_baseline_v1_submission.csv" `
  --cat-oof "C:\Users\mal03\Downloads\result\oof_v22_lite_final.npy" `
  --cat-test "C:\Users\mal03\Downloads\result\test_v22_lite_final.npy" `
  --sample-sub "C:\Users\mal03\Desktop\레퍼런스\sample_submission.csv" `
  --output ".\rank_ensemble_v22_lite_2seed_champ_weight_rebuilt.csv" `
  --verify-against ".\rank_ensemble_v22_lite_2seed_champ_weight.csv"
```

## 검증 기준
- 생성된 CSV의 행 수가 제출본과 동일해야 합니다.
- `ID` 컬럼이 완전히 동일해야 합니다.
- `probability` 값이 제출본과 완전히 동일해야 합니다.
