# 모델 구성 (반드시 이 구조로 구현)
## Model A: Full CatBoost
- 전체 데이터로 학습
- CatBoost native categorical 사용
- 기본 파라미터는 v15 best params를 사용
- 5-fold CV
- 기본 seed ensemble은 5-seed
- 단, runtime 때문에 쉽게 바꿀 수 있도록 CONFIG로 조절 가능하게 해라
## Model B: Transfer-only CatBoost
- subgroup: is_transfer == 1
- 이식 그룹만 사용해 5-fold 학습
- 목적은 이식 그룹 내 ranking 개선
- seed ensemble 지원
- 파라미터는 Full CB와 유사하게 시작하되, transfer subset 전용 설정 가능하게 해라
- 선택적으로 transfer subset 전용 간단 Optuna(예: 10~15 trials, 3-fold) 옵션도 CONFIG로 구현해라. 기본은 False.
## Model C: Non-transfer-only CatBoost
- subgroup: is_transfer == 0
- 비이식 그룹만 따로 학습
- 이 그룹은 이미 쉽기 때문에 Full CB 보정보조 역할
- 비교적 가볍게 돌 수 있게 seeds/config 분리
## Model D: Transfer-only LightGBM
- subgroup: is_transfer == 1
- LightGBM 설치 시 사용
- 없으면 자동으로 XGBoost fallback
- 이 모델은 CatBoost와 다른 편향을 주어 블렌딩 상관을 낮추기 위한 목적
- 반드시 OOF 예측과 test 예측 생성
- categorical은 native LGBM category 대신 안전하게 수치 인코딩/OOF TE 기반으로 처리
- train only fit / fold-safe TE 원칙 준수
## subgroup 정의
- is_transfer = (이식된 배아 수 > 0).astype(int) 를 기본으로 사용
- 결측은 0으로 처리 후 판단
- 로그에 이식/비이식 건수, 양성률을 기록해라

- 5-fold StratifiedKFold
- stratify key는 단순 y만 쓰지 말고 아래 조합을 factorize해서 써라: y + is_transfer + 시술 유형 + 시술 당시 나이
- 즉, strat_key = y.astype(str) + "" + is_transfer.astype(str) + "" + 시술 유형.astype(str) + "_" + 시술 당시 나이.astype(str) factorize 후 StratifiedKFold에 사용
- 각 fold마다:
    - 전체 AUC
    - transfer subgroup AUC
    - non-transfer subgroup AUC
    - fold 학습 시간 기록

## 블렌딩 (매우 중요)
OOF 예측을 기반으로 아래를 모두 비교해라.
1) probability blend
- transfer 구간: pred_transfer = afull_cb + btransfer_cb + c*transfer_lgb (a+b+c=1)
- non-transfer 구간: pred_nontransfer = dfull_cb + enontransfer_cb (d+e=1)
2) raw-score z-blend
- CatBoost raw margin / LightGBM raw score 사용 가능하면 사용
- 각 모델 raw score를 OOF 기준 z-score 정규화 후 가중합
- test에도 train OOF 평균/표준편차 기반으로 동일 변환 적용
3) rank-blend
- 각 예측을 percentile rank 변환 후 평균/가중합

### 블렌딩 탐색 방식
- transfer weights는 0.0~1.0 범위 0.05 step grid search
- non-transfer weights는 0.0~1.0 범위 0.02 step grid search
- 각 blend 방식별 전체 OOF AUC, transfer AUC, non-transfer AUC 계산
- 가장 좋은 최종 방식을 선택
- 선택된 방식/가중치를 로그에 표로 남겨라
